除了前面提到的 `g` 和 `m` 指令，Vim 中还有许多高效的核心指令。掌握它们能极大提升你的编辑速度。我将它们分类整理如下，你可以按需学习和练习。

## 🚀 最核心的高频指令（必须掌握）

### 基础移动
| 指令              | 描述                  | 记忆技巧              |
| :-------------- | :------------------ | :---------------- |
| `h` `j` `k` `l` | 左、下、上、右移动           | 基本方向键             |
| `w`             | **跳到下一个单词开头**       | **w**ord          |
| `b`             | **跳回上一个单词开头**       | **b**ack          |
| `e`             | **跳到当前单词末尾**        | **e**nd           |
| `0`             | 跳到**行首**            | 数字零，最开头           |
| `^`             | 跳到行**第一个非空白字符**     | `^` 在正则中表开头       |
| `$`             | 跳到**行尾**            | `$` 在正则中表结尾       |
| `f{char}`       | 当前行**向前查找字符并跳转**    | **f**ind          |
| `t{char}`       | 当前行向前查字符，跳到**它前面**  | **t**o (the char) |
| `F`/`T`         | 当前行向后查找（反向 `f`/`t`） | 大写即反向             |
| `%`             | 在配对的括号间跳转           | 百分比，配对            |
| `v%`            | 选中括号内的所有内容（包括括号本身）  | 组合记忆（y、d、v、c）     |

### 编辑操作
| 指令                  | 描述            | 记忆技巧                |
| :------------------ | :------------ | :------------------ |
| `i`                 | 在光标**前**插入    | **i**nsert          |
| `a`                 | 在光标**后**插入    | **a**ppend          |
| `I`                 | 在**行首**插入     | 大写的行首插入             |
| `A`                 | 在**行尾**插入     | 大写的行尾追加             |
| `o`                 | 在**下方**新开一行插入 | **o**pen line below |
| `O`                 | 在**上方**新开一行插入 | 大写的上方               |
| `x`                 | 删除光标下字符（剪切）   | 像划掉                 |
| `d{motion}`         | **删除**指定范围    | **d**elete          |
| `c{motion}`         | **删除并进入插入模式** | **c**hange          |
| `y{motion}`         | **复制**指定范围    | **y**ank            |
| `p`                 | 在光标**后**粘贴    | **p**ut after       |
| `P`                 | 在光标**前**粘贴    | 大写的在前               |
| `u`                 | **撤销**        | **u**ndo            |
| `Cmd+z` \| `Ctrl+r` | **重做**        | **r**edo            |
| `.`                 | **重复上一次修改**   | 英文句号，再来一次           |

### 组合示例
```vim
dw     " 删除一个单词（delete word）
ciw    " 删除当前单词并插入（change inner word）
dt)    " 删除直到遇到 ')'（delete to ) // 只在当前行有效
yap    " 复制整个段落（yank a paragraph）
```

## 🔄 操作符 + 动作 = 强大组合
Vim 的精髓是 **操作符(Operator) + 动作(Motion)**。掌握后可以随意组合。

**常用操作符：**
- `d` - 删除
- `c` - 修改（删除并插入）
- `y` - 复制
- `>` - 增加缩进
- `<` - 减少缩进
- `gU` - 转大写
- `gu` - 转小写

**常用文本对象（Text Objects）：**

| 对象 | 描述 | 示例 |
| :--- | :--- | :--- |
| `iw` | **内层单词**（不含周围空格） | `diw` 删除单词 |
| `aw` | **整个单词**（含周围空格） | `daw` 删除单词及空格 |
| `i"` | **引号内的内容** | `ci"` 修改引号内文本 |
| `a"` | **引号及内容** | `da"` 删除引号和内容 |
| `i(` 或 `ib` | **圆括号内** | `ci(` 修改括号内 |
| `a(` 或 `ab` | **圆括号及内容** | `da(` 删除括号及内容 |
| `i{` 或 `iB` | **花括号内** | `ci{` 修改花括号内 |
| `a{` 或 `aB` | **花括号及内容** | `da{` 删除花括号及内容 |
| `ip` | **段落内** | `dip` 删除段落 |
| `ap` | **段落及空行** | `dap` 删除段落 |

## 📁 文件与窗口管理

### 分屏操作
| 指令               | 描述                         |
| :--------------- | :------------------------- |
| `:sp {file}`     | **水平分割**窗口（split）          |
| `:vsp {file}`    | **垂直分割**窗口（vertical split） |
| `Ctrl+w h/j/k/l` | 在窗口间**移动焦点**               |
| `Ctrl+w w`       | 循环切换窗口                     |
| `Ctrl+w =`       | 使所有窗口**等宽等高**              |
| `Ctrl+w _`       | 当前窗口**最大化高度**              |
| `Ctrl+w \|`      | 当前窗口**最大化宽度**              |
| `Ctrl+w c`       | **关闭**当前窗口（close）          |
| `Ctrl+w o`       | **只保留**当前窗口（only）          |

### 标签页
| 指令               | 描述                    |
| :--------------- | :-------------------- |
| `:tabnew {file}` | 在新标签页打开文件             |
| `gt`             | 切换到**下一个**标签页（go tab） |
| `gT`             | 切换到**上一个**标签页         |
| `{N}gt`          | 切换到第 N 个标签页（如 `3gt`）  |

### 文件操作
| 指令               | 描述              |
| :--------------- | :-------------- |
| `:e {file}`      | **打开**文件（edit）  |
| `:w`             | **保存**文件（write） |
| `:q`             | **退出**（quit）    |
| `:wq` 或 `ZZ`     | 保存并退出           |
| `:q!` 或 `ZQ`     | **强制退出不保存**     |
| `:saveas {path}` | 另存为             |

## 🔍 搜索与替换

### 搜索
| 指令               | 描述                 |
| :--------------- | :----------------- |
| `/{pattern}`     | **向下搜索**模式         |
| `?{pattern}`     | **向上搜索**模式         |
| `n`              | 跳转到**下一个**匹配       |
| `N`              | 跳转到**上一个**匹配       |
| `*`              | ==搜索当前光标下的**单词**== |
| `#`              | ===反向搜索当前单词==      |
| `:%s/old/new/g`  | **全局替换**（每行所有匹配）   |
| `:%s/old/new/gc` | 全局替换，**每次确认**      |

### 替换示例
```vim
:s/foo/bar/g      " 当前行替换
:5,10s/foo/bar/g  " 5-10行替换
:'<,'>s/foo/bar/g " 在可视选择区域替换
:%s/foo/bar/gc    " 全文替换，每次确认
```

## ⚡ 进阶高效技巧

### 数字增强
几乎所有指令都支持数字前缀：
```vim
3w          " 向前跳3个单词
5j          " 向下跳5行
2dd         " 删除2行
y3e         " 复制到第3个单词末尾
10Ctrl+f    " 向下翻10页
```

### 寄存器使用
| 指令 | 描述 |
| :--- | :--- |
| `"{reg}y` | 复制到指定寄存器（如 `"ayy` 复制行到寄存器 a） |
| `"{reg}p` | 粘贴指定寄存器的内容（如 `"ap`） |
| `:reg` | 查看所有寄存器内容 |
| `"+y` | 复制到系统剪贴板（需支持） |
| `"+p` | 从系统剪贴板粘贴 |

### ==宏录制==
```vim
qa          " 开始录制宏到寄存器 a
...（操作）...
q           " 停止录制
@a          " 执行宏 a
@@          " 重复执行上次宏
```

## 🛠️ VS Code Vim 特有技巧

在 VS Code 中，除了 Vim 指令，还可以结合编辑器功能：

1. **Ctrl+P**：快速打开文件（覆盖 Vim 的 `:e`）
2. **Ctrl+Shift+P**：命令面板（万能）
3. **F12** / **gd**：跳转到定义
4. **Shift+F12**：查看引用
5. **Ctrl+.**：快速修复
6. **Alt+↑/↓**：上下移动行（比 Vim 的 `ddp` 直观）

### VS Code 集成命令
```vim
:VSCodeCall  " 调用 VS Code 命令
:VSCodeNotify
" 例如：在 .vimrc 中映射
nnoremap <C-s> :VSCodeCall workbench.action.files.save<CR>
```

## 💡 学习路径建议

1. **第一周**：掌握 `h/j/k/l`、`i`、`Esc`、`:wq`、`dd`、`yy`、`p`
2. **第二周**：掌握 `w`、`b`、`e`、`0`、`$`、`f`、`t`、`u`、`Ctrl+r`
3. **第三周**：掌握 `d{motion}`、`c{motion}`、`y{motion}` 组合
4. **第四周**：掌握文本对象 `iw`、`i"`、`i(` 等
5. **长期**：练习宏、寄存器、窗口管理

## 📚 练习方法
1. **vimtutor**：终端输入 `vimtutor` 进行官方教程
2. **VSCode Vim 沙盒**：新建文件专注练习
3. **真实编码**：每天强迫自己用 1-2 个新指令
4. **打印速查表**：贴在显示器旁边随时查看

最值得深入练习的是 **操作符+动作组合** 和 **文本对象**，它们能让你用很少的按键完成复杂编辑。你目前在编码中最常重复的操作是什么？我可以告诉你对应的 Vim 快捷方式。
