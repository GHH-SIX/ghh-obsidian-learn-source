# MetaJump 插件使用说明

## 插件简介
MetaJump 是一款强大的VSCode光标快速移动与编辑插件。它能让你通过简单的键盘指令，将光标瞬间移动到屏幕上的任何可见字符处，同时支持快速选择、删除和添加多光标操作，特别适合追求键盘流操作效率的开发者。

> **注**：MetaJump 由开发者利用业余时间维护，如果你觉得它很棒，可以在GitHub上给个⭐或通过Sponsor支持。

## 核心特性与优势
- **智能字符编码**：根据键入难度为候选位置分配编码字符（如 `k`、`j` 优先级高），编码可配置。
- **涟漪编码 (Ripple Encoding)**：根据输入字符数量逐步扩大搜索范围，减少屏幕上的干扰标记。
- **全面兼容**：完美支持Vim插件，兼容多光标、选区操作，支持在所有已打开的编辑器标签页间跳转。
- **高效精准**：仅编码当前屏幕可见区域内的目标，响应迅速。
- **灵活目标**：不仅限于字母，`Tab`、`空格` 和 `回车` 也可以作为目标位置字符。

## 一、基本跳转：去任何地方
这是插件的核心功能，让你将光标瞬间移动到屏幕上的任意字符。

### 操作步骤
1.  **触发**：按下 `Alt + /`（默认快捷键）。
2.  **输入目标字符**：输入屏幕上目标位置附近的字符（单词的一部分）。MetaJump 会为可能的候选位置**叠加显示编码字符**。
3.  **完成跳转**：输入你想要到达的那个候选位置上的**编码字符**，即可瞬间跳转。

### 操作技巧与说明
*   **查看原字符**：输入过程中，可以按 `Shift+/`（可配置）**临时隐藏**位置标记，查看被覆盖的原始字符约560毫秒。
*   **筛选目标**：在输入目标字符后，可以**继续输入后续字符**来缩小候选范围，让编码更精确。
*   **取消与回退**：任何时候按 `Esc` 可取消整个命令；按 `Backspace` 可回退上一步输入。
*   **三种跳转模式**：
    *   `Alt + .`：跳转到**目标字符之后**。
    *   `Alt + ,`：跳转到**目标字符之前**。
    *   `Alt + /`（智能跳转）：如果目标位于单词开头，光标停在字符**前**；否则停在字符**后**。

### 理解“涟漪编码”
为了减少标记数量，插件引入了“涟漪编码”概念。它的工作原理就像向水中投入石子，波纹由近及远扩散：
*   **输入1个目标字符**：仅在当前“节”（由空行分隔的代码块）内编码候选位置。
*   **输入2个目标字符**：在整个当前文档内编码。
*   **输入3个或更多字符**：在所有已打开的编辑器中进行全局编码。

**实用跳转技巧**：
*   想跳转到**同一文件但不同节**，直接输入**2个**目标字符。
*   想跳转到**另一个已打开的标签页文件**，直接输入**3个**目标字符。

## 二、高级编辑操作
除了跳转，MetaJump 还提供了一系列基于相同机制的强大编辑命令。

### 1. 快速选择
*   **触发**：`Alt + Shift + /`
*   **操作**：与跳转类似，输入目标字符后，再输入候选位置的编码字符，即可**从当前光标选择到目标位置**。此操作可重复进行，以调整选区范围。

### 2. 添加多光标 / 变更活动选区
这组命令对于批量编辑极其高效。
*   **添加光标（在目标前）**：`Ctrl + Alt + ,`
*   **添加光标（在目标后）**：`Ctrl + Alt + .`
*   **智能添加光标**：`Ctrl + Alt + /`
*   **取消上一个光标**：`Ctrl + u`

**核心技巧**：当目标位置**位于一个已有选区内**时，上述三个命令会变为“**变更活动选区**”命令。这允许你轻松调整多个选区中哪一个处于活动状态（可以扩展或收缩其范围），是精细调整多重选择范围的利器。

### 3. 快速删除
*   **智能删除至目标**：`Alt + d`（删除从光标到目标位置，智能判断前后）
*   **删除至目标前**：`Alt + Backspace`
*   **删除至目标后**：`Alt + Delete`

## 三、命令总结与配置提示
为了方便查阅，以下是核心命令的默认快捷键汇总：

| 功能 | 默认快捷键 | 对应命令 |
| :--- | :--- | :--- |
| **智能跳转** | `Alt + /` | `metaGo.gotoSmart` |
| **跳转到目标后** | `Alt + .` | `metaGo.gotoAfter` |
| **跳转到目标前** | `Alt + ,` | `metaGo.gotoBefore` |
| **快速选择** | `Alt + Shift + /` | `metaGo.selectSmart` |
| **智能添加光标** | `Ctrl + Alt + /` | `metaGo.addCursorSmart` |
| **智能删除** | `Alt + d` | `metaGo.metaJump.deleteToSmart` |

### 自定义配置
你可以在VSCode的设置 (`settings.json`) 中自定义插件行为：
*   **修改快捷键**：通过命令面板查找上述命令，并分配新的键位。
*   **调整编码行为**：例如，可以通过设置 `"metaGo.jumper.commandsToUseSingleTargetChar"` 来指定哪些命令使用“单字符目标”模式（输入一个字符即显示所有候选编码）。
